신규 기능 1
빈 셀 그리드 배경 — 시도와 해결
시도했으나 실패한 방법
  • background-attachment: local
    콘텐츠 영역에만 패턴 적용 — 빈 공간에 안 보임
  • CSS 변수(var(--cell-border))를 gradient 색상 스톱에 사용
    CSS 변수는 gradient 내 색상 스톱에 미작동
최종 해결 — gradient 하드코딩
  • 행 구분선: repeating-linear-gradient 52px 간격
  • 열 구분선: linear-gradient 고정 px stops
  • 컬럼 경계 stops: 42 / 152 / 432 / 528 / 868px
  • background-size: 100% 52px, 100% 100%
CSS 변수 gradient 한계로 인해 컬럼 너비 값을 직접 하드코딩. 컬럼 너비 변경 시 함께 수정 필요.
