구현

데모 구현 현황

Demo A — Particle Flow

  • p5.js instance mode + pixelDensity(1)
  • sin/cos 플로우 필드 (경량화)
  • blendMode(ADD) 글로우
  • 파티클 80–400개 적응형 조절
  • FPS <28 시 자동 감소 / >52 시 증가

Demo B — Neon Tunnel

  • @react-three/fiber + @react-three/drei
  • 16개 링 (6·8·10각형 교차)
  • 10초 주기 팔레트 자동 전환
  • 이코사헤드론 + 옥타헤드론 중심
  • 마우스 기반 카메라 lerp

Demo C — Audio Reactive

  • useAudioAnalyzer — 음악/탭/음성 3모드
  • 음악: 파일 피커 → 주파수 바 비주얼
  • 탭: getDisplayMedia 시스템 오디오
  • 음성: 마이크 + 노이즈 게이트 (RMS >0.015)
  • 음성 모드: 오실로스코프 + 에너지 링
공통 기능: 전체화면 API 마우스 이벤트 50ms 반응 useFrameRate FPS 모니터링 FpsOverlay 색상 코딩 1/2/3 단축키 전환 React.lazy 코드 스플리팅
