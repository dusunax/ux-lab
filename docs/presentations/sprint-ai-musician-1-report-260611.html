ARCHITECTURE

페르소나에서 MP4까지 단일 앱 흐름으로 연결

Product Flow

1PersonaForm에서 뮤지션 정체성·장르·작가관 입력
2PromptPreview에서 트랙 스토리 기반 가사/프롬프트 생성
3TrackList와 PlayerBar에서 결과 트랙 관리·재생
4exportAlbum으로 JSON 다운로드
5VideoRender에서 공유용 MP4 생성

Render Flow

API/api/render-video multipart 입력 파싱
I/O오디오·이미지 URL 또는 파일을 temp로 저장
TXT@napi-rs/canvas로 한글 텍스트 레이어 생성
MP4ffmpeg scale/pad/overlay/libx264로 합성
OBSrequest id 기반 단계 로그와 abort 처리
