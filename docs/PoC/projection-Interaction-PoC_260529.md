# 인터랙티브 프로젝션 아트 데모 스프린트 기획서

# 1. 배경

프로젝터를 활용한 공간 연출은 단순 영상 재생을 넘어, 사용자의 입력과 실시간으로 반응하는 인터랙티브 경험으로 확장되고 있다.

이번 스프린트에서는 프로젝터와 PC 환경을 기반으로, 마우스 입력에 반응하는 실시간 비주얼 시스템을 구현하여 인터랙티브 미디어아트의 기술 가능성을 검증한다.

본 작업은 단순 시각 효과 구현이 아닌, “공간 자체가 반응하는 경험”을 만드는 것을 목표로 한다.

---

# 2. 스프린트 목표

## 핵심 목표

* 프로젝터 기반 실시간 인터랙션 데모 구현
* 마우스 입력 기반 비주얼 반응 시스템 구축
* 몰입형 공간 연출 가능성 검증
* 향후 프로젝션 매핑 및 센서 인터랙션 확장 기반 확보

# 5. 기대 사용자 경험

## 목표 경험

사용자는 단순히 영상을 시청하는 것이 아니라, 자신의 입력에 따라 공간 전체가 반응하는 경험을 체감한다.

다음과 같은 감성적 경험을 목표로 한다.

* 빛을 공간 위에 그리는 느낌
* 벽면이 살아 움직이는 느낌
* 음악과 공간이 동기화되는 느낌
* 몰입형 디지털 공간 체험

# 7. 구현 대상 데모

## Demo A — Particle Flow

마우스 움직임을 따라 입자와 빛이 생성되고 공간 전체로 확산되는 인터랙션 구현

### 검증 포인트

* 입력 반응성
* 파티클 렌더링 성능
* 공간 몰입감

---

## Demo B — Neon Tunnel

사용자 입력에 따라 시점과 공간 왜곡이 변화하는 네온 기반 터널 비주얼 구현

### 검증 포인트

* 3D 공간감
* GPU 렌더링 성능
* 프로젝터 몰입도

## Demo C — Audio Reactive Visual

### 검증 포인트

* 오디오 분석 안정성

* 감성적 연출 효과
