# AI 뮤지션 앱 PoC 기획서

# 1. 배경

플레이리스트에서 AI 음악의 비중이 높아졌지만, AI 음악은 "테마/제목"만 존재하고 일관된 작가관이 없다.

한 앨범·한 가수의 세계관을 중점적으로 듣기를 좋아하는 사람에게는 매력이 떨어진다.
마치 잘 그린 그림 한 장처럼, 단독으로 보면 훌륭하지만 작가관이 없으면 예술적 깊이가 없다.

이번 PoC에서는 작가관을 가진 AI 뮤지션 페르소나를 정의하고, 그 페르소나 기반으로 음원을 생성하는 파이프라인을 검증한다.

---

# 2. 스프린트 목표

## 핵심 목표

* 작가관을 가진 AI 뮤지션 페르소나 정의 시스템 구현
* 페르소나 기반 음원 생성 파이프라인 검증
* 앨범 단위 메타데이터 구조 설계 및 출력
* 페르소나 교체 시 스타일 변환 가능성 검증

---

# 3. 구현 범위

## 포함 범위

### 페르소나 시스템

* AI 뮤지션 페르소나 정의 UI
* 페르소나 데이터 구조 (JSON/구조화 데이터)
* 페르소나 교체 및 스타일 변환

### 음원 생성

* AI 음악 생성 API 연동 (Suno AI, Udio 등)
* 프롬프트 엔지니어링 기반 음원 생성
* 생성된 트랙 미리듣기

### 앨범 구성

* 트랙 목록 구성 (순서, 제목, 설명)
* 앨범 메타데이터 JSON 관리
* 커버아트 생성 (AI 이미지 생성 연동)

### 출력 포맷

* YouTube 업로드용 (영상+음원)
* 앨범 메타데이터 JSON 다운로드

---

# 4. 제외 범위

* 실제 스트리밍 플랫폼 배포 (Spotify, SoundCloud 등)
* 공개 페르소나 갤러리
* 팬 커뮤니티 기능
* AI 뮤지션 성장 시스템

---

# 5. 기대 사용자 경험

## 목표 경험

사용자는 페르소나를 정의하는 것만으로 "이 뮤지션이 만든 앨범"이라는 감각을 체험한다.

다음과 같은 감성적 경험을 목표로 한다.

* 같은 UI에서 페르소나만 바꾸면 완전히 다른 스타일의 음악이 생성되는 경험
* 단순한 AI 생성곡이 아닌 한 아티스트의 세계관으로 느껴지는 경험
* 앨범 커버·트랙 순서·메타데이터가 하나의 작품으로 통합되는 경험

---

# 6. 기술 방향

## 기술 스택

| 영역          | 기술                              |
| ----------- | ------------------------------- |
| 프레임워크       | Next.js App Router, TypeScript  |
| 모노레포        | ux-lab monorepo                 |
| 음원 생성 API   | Suno AI, Udio (프롬프트 엔지니어링 기반)   |
| 이미지 생성      | AI 이미지 생성 API (커버아트)            |
| 페르소나 데이터 구조 | JSON / 구조화 데이터                  |
| 출력 포맷       | YouTube 업로드용 (영상+음원), JSON 다운로드 |

---

# 7. 구현 대상 데모

## Demo A — 페르소나 정의 및 음원 생성 플로우

뮤지션 페르소나 입력 UI에서 프롬프트를 생성하고 AI 음악 생성 API와 연동하여 트랙을 생성한다.

### 검증 포인트

* 페르소나 → 프롬프트 변환 정확성
* AI 음악 생성 API 연동 안정성
* 생성 트랙 미리듣기 동작

---

## Demo B — 앨범 구성 및 출력

생성된 트랙을 앨범 단위로 묶고 커버아트를 생성하여 메타데이터 JSON으로 출력한다.

### 검증 포인트

* 앨범 메타데이터 구조 완결성
* 커버아트 생성 연동 안정성
* JSON 출력 포맷 적합성

---

## Demo C — 페르소나 교체 스타일 변환

동일한 감성·주제를 다른 페르소나로 재해석하고 페르소나별 출력을 비교한다.

### 검증 포인트

* 페르소나 교체 시 음악 스타일 차별성
* "같은 주제, 다른 세계관" 경험 전달 여부
* 비교 UI 사용성

# 9. 성공 기준

## 기술 기준

* 페르소나 정의 → 프롬프트 생성 → 음원 생성 파이프라인이 동작할 것
* 앨범 단위로 트랙 메타데이터를 저장하고 출력할 수 있을 것

## 경험 기준

* 같은 UI에서 페르소나만 바꾸면 완전히 다른 스타일의 음악이 생성될 것
* "이 뮤지션이 만든 앨범"이라는 감각이 생길 것
