# PoC 문서 인덱스

외부 인사 또는 협업 클라이언트와 함께 진행하는 PoC 문서는 일반 스프린트 회의록과 분리해 관리한다.

## 보관 규칙

- 경로: `docs/PoC/도메인-협업클라이언트명/`
- 예시: `docs/PoC/projection-art-이하나/`
- 스프린트 내부 회의록은 `docs/meetings/`에 두고, 클라이언트 인터뷰·협업 설계 회의·외부 PoC 논의는 `docs/PoC/`에 둔다.
