# docs/ai/

## 폴더 목적

AI 프롬프트 문서, 응답 샘플 기준선, 모델 선택 근거를 보관한다.

- 프롬프트 변경 이력을 추적하고 회귀 기준을 유지하기 위한 문서 저장소다.
- QA가 기준선 샘플과 신규 응답을 비교할 때 이 폴더를 참조한다.
- PM + UX가 프롬프트 개선안을 승인할 때 이 폴더의 문서를 검토 대상으로 삼는다.

---

## 업데이트 규칙

1. **프롬프트를 변경할 때는 새 파일을 생성한다.** 기존 파일을 덮어쓰지 않는다.
   - 예: `prompt-review-v1.md` → `prompt-review-v2.md`
2. **버전 번호는 단조 증가한다.** `v1`, `v2`, `v3` 순서로만 올린다. 건너뛰지 않는다.
3. **기준선 샘플은 프롬프트 버전별로 구분한다.** 파일 내 섹션으로 분리하거나 별도 파일(`baseline-samples-v2.md`)로 관리한다.
4. **이 README의 파일 목록을 함께 업데이트한다.** 새 파일 추가 시 표에 행을 추가한다.
5. **모델이 변경될 때도 버전을 올린다.** 프롬프트 텍스트 변경과 동급의 변경 사항으로 간주한다.
