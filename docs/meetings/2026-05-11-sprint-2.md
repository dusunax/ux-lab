## 논의 내용

### 1. Avery 면담 — 버그 수정

유저가 직접 발견한 4가지 문제:

| 문제 | 원인 | 해결 |
|------|------|------|
| col-B 텍스트 가로 잘림 | `.cell` 기본 `white-space: nowrap` 미 오버라이드 | col-b/d에 `white-space: normal; word-break: break-word` |
| 스켈레톤 색이 너무 진함 | `#aaa` → `#d4d4d4` 둘 다 부적절 | shimmer 애니메이션 바(`#e0e0e0 → #f0f0f0`)로 교체 |
| IME "딩" 잔류 (Mac) | 한글 IME가 `input.value = ''` 후 조합 완성 문자 삽입 | `e.isComposing \|\| e.keyCode === 229` 가드 |
| row 배경 투명 | `.data-row`에 `background-color` 미지정 | `background-color: var(--cell-bg)` 추가 |

### 2. 빈 공간 셀 그리드 배경

**요건:** 마지막 행 아래 빈 공간에 엑셀 빈 셀처럼 보이는 패턴

**시도 및 실패:**
- `background-attachment: local` — 콘텐츠 영역에만 패턴 적용돼 빈 공간에서 안 보임
- `var(--cell-border)` in gradient — CSS 변수 gradient 내 색상 스톱 미작동

**최종 구현:**
```css
.sheet-body {
  background-image:
    repeating-linear-gradient(to bottom, transparent 0px, transparent 51px, #d0d0d0 51px, #d0d0d0 52px),
    linear-gradient(to right, #d0d0d0 1px, transparent 1px, ...);
  background-size: 100% 52px, 100% 100%;
}
```
컬럼 구분선 stops: 42px / 152px / 432px / 528px / 868px (각 컬럼 너비 기준)

### 3. 감정 필터 (AutoFilter 스타일)

**설계 (Riley):** 엑셀 AutoFilter 드롭다운 재현. 활성 시 ▼ → ▾ + 배지 색상 표시.

**구현 내용:**
- 필터 헤더: `.filter-header > .filter-trigger` 클릭 시 `.filter-dropdown.open`
- 옵션: `전체` + 13개 감정 (각 `EMOTION_CONFIG` 배지 색상 적용)
- 활성 필터: trigger에 활성 감정 배지 색 표시
- 결과 없음: `.no-results` 안내 메시지
- 외부 클릭으로 드롭다운 닫힘

**Quinn 발견 버그 3건 수정:**

| 버그 | 수정 |
|------|------|
| 필터 활성 중 로딩 행 삽입 시 필터 무시됨 | `prependLoadingRow`: `if (virtualActive \|\| filterEmotion)` → `renderRows()` |
| AI 결과 도착 시 필터된 행 DOM 업데이트 안됨 | `applyAnalysisResult`: 동일 조건 추가 |
| 드롭다운 옵션 클릭 후 즉시 재열림 | 옵션 클릭에 `e.stopPropagation()` 추가 |

### 4. 삭제 버튼

**설계 (Riley):** col-E 우측, hover 시 fade-in ✕ 버튼. 클릭 시 fade-out 후 삭제.

**구현:**
- `.delete-btn { opacity: 0; transition: opacity 0.15s }`
- `.data-row:hover .delete-btn { opacity: 1 }`
- `@keyframes rowFadeOut { to { opacity: 0; transform: translateX(8px); } }`
- `handleDeleteRow()`: `row-removing` 클래스 추가 → `animationend` 후 entries 배열에서 제거 → `renderRows()`
