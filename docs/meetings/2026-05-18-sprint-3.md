## 논의 내용

### 1. ai-empathy-diary 서비스 품질 다듬기

Sprint 2까지 완성된 기능(감정 필터, 삭제 버튼, 빈 셀 그리드 배경)을 기반으로 실제 서비스 수준으로 다듬는다.  
UX 개선, 에러 처리, 빈 상태(empty state) 대응 등이 포함된다.

### 2. 웹 배포: Vercel

- 배포 대상: ai-empathy-diary
- 플랫폼: Vercel (MCP 설치 완료)
- 환경 변수 관리 및 프리뷰 배포 전략 포함

### 3. Auth 구현: Firebase Google 로그인 OAuth

- Firebase Authentication 사용 (Firebase MCP 설치 완료)
