## 클라이언트 핵심 요구사항 요약

이하나 작가의 발언을 기반으로 도출한 요구사항.

### 전반적 방향

- 현재 Demo E의 "차갑고 우주적인" 인상을 "살아있는 바닷속" 인상으로 전환
- 클라이언트의 동화책 세계관 — 제주 바다, 야광충, 깊이감 있는 수중 — 을 기준점으로 삼음
- 아이들을 대상 관객으로 상정한 감성적 완성도 요구

### 배경 및 빛

- 단순 검정 배경 불가. 파랑이 도는 어둠으로 대체
- 깊이에 따른 수층 구분 — 수온약층처럼 경계가 흐릿하게 느껴지는 그라디언트
- 빛은 파도와 함께 흔들려야 함. 고정된 빛은 비사실적으로 느껴짐
- 물속 부유 입자(플랑크톤, 미세 모래)가 빛을 산란시키는 표현 필요

### 파도 레이어

- 수면선은 단일 선이 아닌 4~5겹 레이어 (앞쪽/뒤쪽 파도 + 반투명 물)
- 수면 위치: 화면 상단 1/3 기준 → 깊은 바닷속 느낌
- 각 레이어 다른 속도로 움직여 깊이감 부여
- 파도 진폭이 신체 모션 에너지에 연동되어 동적으로 반응

### 신체 표현

- 스켈레톤 라인 스타일 제거. 해파리 촉수처럼 반투명 발광하는 흐름으로 대체
- 움직임 반응이 즉각적이지 않아야 함 — 수중 저항감 있는 지연
- 정지 후 잔상이 천천히 소멸

### 파티클(플랑크톤)

- 평소: 천천히 위로 부유
- 손 접근 시: 폭발적 반응 아닌 물 밀리듯 완만하게 분산
- 밀도와 반응 강도 모두 절제 — 과잉 연출 명시적으로 거부

### 생물발광

- 야광충 발광 표현 포함 — 색상 `#80ffdb` (민트-청록)
- 사용량은 극소량 유지. 과다 사용 시 형광등 느낌으로 변질

### 파도 레이어 스펙

| 속성 | 값 |
|------|-----|
| 레이어 수 | 4~5겹 |
| 수면 위치 | 화면 상단 1/3 기준 |
| 레이어별 속도 | 독립 설정 (앞쪽 빠름 / 뒤쪽 느림) |
| 진폭 연동 | 모션 에너지 값에 실시간 연동 |
| 레이어 투명도 | 뒤쪽 레이어는 반투명 처리 |
| 빛 연동 | 파도 움직임에 따라 수중 빛 위치 동기화 |

| 속성 | 값 |
|------|-----|

| 빛 산란 | 부유 입자가 광원을 통과하며 반짝이는 표현 |
| 생물발광 빈도 | 극소량 — 과잉 시 형광등 효과로 변질 |

| 속성 | 값 |
|------|-----|
