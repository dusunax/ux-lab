# 달력한칸 (one-moon-date)

Android 홈 화면에서 오늘의 음력 날짜를
달력의 한 칸처럼 보여주는 React Native 앱

---

## 📛 프로젝트 정보

- 앱 이름: 달력한칸
- 영문 이름: one-moon-date
- 플랫폼: Android
- 프레임워크: React Native

---

## 🧩 앱 컨셉

사용자가 앱을 열지 않아도  
안드로이드 홈 화면에서 오늘의 음력 날짜를 한눈에 확인할 수 있다.

핵심 키워드:
- 음력
- 홈 화면
- 한 칸
- 미니멀

---

## 🎯 MVP 요구사항

### 기술 스택
- React Native (Android 전용)
- JavaScript
- Expo 사용하지 않음
- 외부 라이브러리 최소화

---

### 음력 날짜 계산
- 한국 음력 기준
- 음력 월 / 일 표시
- 윤달 처리
- 오프라인 동작

---

### 홈 화면 표시
- Android App Widget 사용
- 표시 내용
  - 크게: 음력 ‘일’
  - 작게: 음력 ‘월’
- 하루 1회 자동 갱신 (자정 기준)

---

## 🎨 UI / UX

- 미니멀
- 차분한 한국 감성
- 타이포그래피 중심
- 단일 화면 구성

---

## 🧱 아키텍처 방향

- 음력 계산 로직 분리
- Android 위젯 네이티브 코드 분리
- React Native UI는 단순하게 유지

---

## 🚫 제한 사항

- Expo 사용 금지
- 불필요한 라이브러리 추가 금지
- Android 제약 사항은 명확히 설명할 것

**가벼운 모델 사용 (단순 작업):**
- 코드 포맷팅 및 스타일 수정
- 작은 버그 수정 (1-2줄 변경)
- 주석 추가/제거
- 변수명 변경
- 간단한 리팩토링 (함수 분리 등)
- 파일 이동/이름 변경

**강력한 모델 사용 (복잡한 작업):**
- 아키텍처 설계 및 큰 리팩토링
- 새로운 기능 추가 (복잡한 로직)
- 여러 파일에 걸친 수정
- 알고리즘 구현
- 복잡한 상태 관리 설계
